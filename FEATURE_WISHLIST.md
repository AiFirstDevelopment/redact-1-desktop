# FOIA Supervisor Feature Wishlist

Features that FOIA supervisors and records managers commonly request, based on industry research.

## 1. Supervisor Review Workflow
- [ ] Approval queue for redactions before release
- [ ] Ability to review and modify redactions made by clerks
- [ ] Deadline reminders and escalation alerts

## 2. Batch/Bulk Operations
- [ ] Process multiple files at once
- [ ] Apply same redaction rules across document sets
- [ ] Bulk export with consistent settings

## 3. Reporting & Analytics Dashboard
- [ ] Request volume trends over time
- [ ] Average processing time per request
- [ ] Staff workload distribution
- [ ] Overdue request alerts

## 4. Cross-Request Search
- [ ] Find previously redacted documents to avoid duplicate work
- [ ] Search across all requests for responsive records
- [ ] Template redactions for recurring PII patterns

## 5. Deadline Tracking
- [ ] Due date field on requests
- [ ] Visual timeline of due dates
- [ ] Automatic reminders (7 days, 3 days, 1 day)
- [ ] Extension request workflow

## 6. Audit Trail Export
- [x] Full audit trail maintained (existing)
- [ ] Exportable logs for legal review
- [ ] Defensible record of all actions

## 7. Video/Audio Redaction
- [ ] Body cam footage support
- [ ] Audio redaction for 911 calls
- [ ] Automated face/license plate detection in video

## 8. Request Portal Integration
- [ ] Requester can check status online
- [ ] Self-service for simple requests
- [ ] Reduces phone/email inquiries
